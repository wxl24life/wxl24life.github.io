Java 按值传递还是按引用传递？

发表于 2019-06-26 | 阅读次数:

先说答案：Java is always pass-by-value。

下面的例子里，将 aDog 变量传递给了 foo 方法，并在其内部对参数做了重新赋值，foo 方法执行完后，aDog 仍然指向了原来的对象。验证了 Java 按值传递的特性。
