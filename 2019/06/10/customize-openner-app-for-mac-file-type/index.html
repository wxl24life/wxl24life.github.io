定制 Mac 文件类型的默认打开程序

发表于 2019-06-10 | 更新于: 2019-06-10

最近开始使用 MacDown 来写博客了。在命令行下通过 Hexo 生成文件之后，习惯使用 open 命令直接打开新生成的文件。本文介绍如何修改 open 命令默认启动的打开程序 app.

例如，在编辑这篇文章时，我的操作步骤是：

$ hexo new post customize-openner-app-for-mac-file-type

INFO Created: ~/practices/hexo-wxl/source/_posts/customize-openner-app-for-mac-file-type.md

紧接着通过 open 命令打开文件：
$ open ~/practices/hexo-wxl/source/_posts/customize-openner-app-for-mac-file-type.md

这里有个问题，open 命令在打开文件时，采用文件类型所指定的默认 app。比如，我之前 .md 文件的默认打开程序是 Ulyssys，所以这里我需要将默认 app 关联到 MacDown 程序。

修改某文件类型默认打开程序的方式

总共 4 个步骤。

  1. 找到任意一个要修改默认打开程序的文件，我这里是 .md 文件
  2. 右键 -> Get Info
  3. 在出来的详细信息页面里找到 Open With，从下拉列表中选择 MacDown.app
  4. 点击下方的 Change ALL，并点击确认即可。

修改单个文件的默认打开程序

下面的操作只影响到单个文件，并不会修改同类型的其他文件。

找到要打开的文件，右键，保持菜单列表显示的状态，按住键盘上的 Option(Alt) 键，点击 Always Open With，之后选择新的打开程序即可。
