Hexo 往 GitHub 发布失败

发表于 2019-07-17 | 阅读次数:

前几天修改了下博客的 About 页面，之后执行 hexo generate & hexo deploy，以为发布成功了。因为终端窗口一直没关闭，每次切到这个窗口都发现有报错，习惯性地认为是网络问题（之前遇到过），重新执行命令。今天早上发现，还是失败了，报错跟之前一样。

仔细看了下报错：

试着执行了一下 ssh -T git@github.com，提示

看上去访问没问题。重新执行 hexo 命令，发布成功。
